# TheCuriousGeek Releases
This repository contains all the releases by TheCuriousGeek
- Certificate Generator - Small utility to generate self signed certificates 
- FileSync - RClone GUI to manage synchronization 
- FileVault - Encrypted vault serving http(s) and WebDAV 
- FileVault Windows - Windows app to encrypt files in a vault
- MediaShelf - Media Manager to manage all media (photos, audio, video and ebooks)
- Home Assistant Addon - A custom component to manage Bluetooth, Tuya and Zigbee devices in Home Assistant
- Notes - App to manage personal notes (including encryption) with attachments
- SimpleKiosk - Android Kiosk app that can be remote controlled with a browser
